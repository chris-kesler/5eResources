Oneshots

These journals log my D&D games as I play them through each session. Some are focused purely on a specific character and the advancement they make, some are game recordings I've uploaded to YouTube. Oneshots and Campaigns, both that I have participated in as Player and as Game Master, can be viewed as well. The preface goes over the preliminary game/character information. The character journals include a session log that has both an in-character journal entry and an overview table that lists xp and loot and such. The game recordings include a brief description, title, and date when possible.
